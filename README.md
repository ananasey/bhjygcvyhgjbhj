# AI-платформа для адаптации новых сотрудников

Кликабельный desktop-прототип B2B SaaS-платформы для обучения, онбординга и адаптации новых сотрудников.

## Быстрый запуск

Откройте файл `index.html` в браузере

или запустите локальный сервер:

```bash
python3 -m http.server 4173
```

После этого откройте в браузере:

`http://localhost:4173`

## Что внутри

- стартовый экран выбора роли
- интерфейс нового сотрудника
- AI-чат
- маршрут адаптации
- тестирование
- кабинет наставника
- карточка сотрудника
- база знаний администратора
- аналитика
- настройки
