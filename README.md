# AI-платформа для адаптации новых сотрудников

Кликабельный прототип веб-платформы для обучения, онбординга и адаптации новых сотрудников.

Заявка Заявка СтС-619106

## Быстрый запуск

1)Установите все файлы

2)Откройте файл `index.html` в браузере


## Что внутри

- стартовый экран выбора роли
- интерфейс нового сотрудника
- AI-чат
- маршрут адаптации
- тестирование
- кабинет наставника
- карточка сотрудника
- база знаний администратора
- аналитика
- настройки
